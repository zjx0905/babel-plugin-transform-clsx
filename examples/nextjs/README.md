# Nextjs

Use `babel-plugin-clsx` in `Nextjs`.

## Install

npm

```bash
npm i --save-dev babel-plugin-clsx
```

yarn

```bash
yarn add --save-dev babel-plugin-clsx
```

pnpm

```bash
pnpm add --save-dev babel-plugin-clsx
```

## Configuration

.babelrc

```json
{
  "presets": [["next/babel"]],
  "plugins": [["clsx", {}]]
}
```
